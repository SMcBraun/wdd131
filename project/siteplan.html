Hawaii Animal Shelter Dashboard

Site Name

Hawaii Animal Shelter Dashboard

This name was chosen because the website will provide a clear overview of animal shelter data across Hawaii. The goal is to help visitors understand shelter needs, adoption trends, and overlooked animals by island.

Site Purpose

The purpose of this website is to provide a simple dashboard that shares animal shelter information in Hawaii. It will include shelter statistics, adoption-related data, island comparisons, and resources that help users understand animal rescue needs without competing with local shelters.

Scenarios

  • Which island has the highest number of animals waiting for adoption?
  • How can I better understand shelter needs across Hawaii before donating or volunteering?

Color Schema

  • Ocean Blue #0b7285: Used for headings, header background, and accents.
  • Soft Sand #f8f1e7: Used for the page background and section backgrounds.
  • Dark Gray #333333: Used for body text.

Typography

  • Arial: Used for body text because it is simple and easy to read.
  • Georgia: Used for headings to give the site a more polished look.
